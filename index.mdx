## What is tract?

tract is Sonos' neural network inference engine. It loads trained models from popular frameworks — ONNX, TensorFlow, and NNEF — optimizes them, and runs them with minimal overhead on your target hardware.

Whether you're running image classification on a Raspberry Pi, wake-word detection on an ARM Cortex-A, or large language model inference on a GPU, tract gives you a single, dependency-light library to do it.

## Getting started

<Steps>
  <Step title="Add tract to your project">
    Add the `tract` crate to your `Cargo.toml` or install the Python package from PyPI.

    <CodeGroup>
    ```toml Cargo.toml
    [dependencies]
    tract = "0.21"
    ```
    ```bash Python
    pip install tract
    ```
    </CodeGroup>
  </Step>
  <Step title="Load a model">
    Use the ONNX or NNEF loader to read your model file and prepare it for inference.

    ```rust
    use tract::prelude::*;

    let model = tract::onnx()?
        .load("mobilenetv2-7.onnx")?
        .into_model()?
        .into_runnable()?;
    ```
  </Step>
  <Step title="Run inference">
    Pass input tensors and collect results.

    ```rust
    let result = model.run([input])?;
    let best = result[0].as_slice::<f32>()?.iter().enumerate()
        .max_by(|a, b| a.1.partial_cmp(b.1).unwrap());
    println!("Predicted class: {:?}", best);
    ```
  </Step>
</Steps>
