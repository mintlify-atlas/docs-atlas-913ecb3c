The `tract` crate is the recommended public-facing facade for building Rust applications with tract. As of version 0.23 it provides a stable, versioned API that shields you from internal churn.

<Note>
  0.23 is the first version with a stable public API facade. Earlier versions exposed internals directly and are not covered here.
</Note>

## Installation

Add the following to your `Cargo.toml`:

```toml Cargo.toml
[dependencies]
tract = "0.23"
anyhow = "1"
```

## Importing the prelude

All public types are re-exported from `tract::prelude`. Start every file with:

```rust
use tract::prelude::*;
```

## Entry point functions

| Function | Returns | Description |
|---|---|---|
| `tract::onnx()` | `Result<Onnx>` | Create an ONNX parser |
| `tract::nnef()` | `Result<Nnef>` | Create a NNEF parser |
| `tract::runtime_for_name(name)` | `Result<Runtime>` | Look up a named hardware runtime (e.g. GPU) |

## Typical workflow

The standard path from a file on disk to inference results:

<Steps>
  <Step title="Create a framework instance">
    Call `tract::onnx()?` or `tract::nnef()?` to get a loader.
  </Step>
  <Step title="Load the model">
    Call `.load("model.onnx")?` to get an `InferenceModel` (ONNX) or `Model` (NNEF).
  </Step>
  <Step title="Convert to a typed model">
    For ONNX, call `.into_model()?` to produce a fully-typed `Model`.
  </Step>
  <Step title="Compile to a runnable">
    Call `.into_runnable()?` to produce an optimized `Runnable`.
  </Step>
  <Step title="Run inference">
    Call `runnable.run([input_tensor])?` and consume the output `Vec<Tensor>`.
  </Step>
</Steps>

```rust
use tract::prelude::*;

fn main() -> anyhow::Result<()> {
    let model = tract::onnx()?
        .load("model.onnx")?
        .into_model()?
        .into_runnable()?;

    // build an input tensor (shape 1√ó3√ó224√ó224, f32)
    let input: Tensor = tract_ndarray::Array4::<f32>::zeros((1, 3, 224, 224))
        .try_into()?;

    let result = model.run([input])?;

    // result[0] is a Tensor; access the raw f32 values:
    let scores = result[0].as_slice::<f32>()?;
    println!("output shape: {:?}", result[0].shape()?);
    println!("first score: {}", scores[0]);

    Ok(())
}
```
